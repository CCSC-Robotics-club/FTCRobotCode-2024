#### 自动阶段目标 Autonomous Objectives
我们的目标是在自动阶段：
- 首先通过板载TensorFlow识别TeamProp位置，正确放置预装的紫色像素
- AprilTag, odometery和惯导模块将黄色像素放置在背景板上
- 通过巡线传感器和后视距离传感器定位像素仓库，夹取两个白色像素。返回背景板放置
- 重复该流程，再次放置两个白色像素
#### 使用的传感器 Sensors Used
- **自动逼近功能**
    1. 前视距离传感器
    2. 哈士奇视觉传感器
    3. 编码器轮
    4. 惯性导航模块
- **自动阶段导航**
    1. 编码器轮
    2. 惯性导航模块
- **自动阶段定位像素堆**
    1. 巡线传感器
    2. 后视距离传感器
#### 关键算法 Key Algorithms
- **从动轮导航系统**
    1. 平行轮组位移测算算法
    2. 最小二乘回归线校准算法
- **自动瞄准系统**
    1. 自动接近
        1. 单目相机线性投射位置计算法
        2. PID纠正算法
    2. 自动贴合
        1. TOF距离传感器距离感知
    3. 无级调整手臂高度
        1. Linear Lookup Table
- **手动阶段精准控制**
    1. 动态轨迹纠正算法与PID

#### 操作控制优化 Driver Controlled Enhancements
-  **自动瞄准系统**
    1. 自动接近：触发自动瞄准后通过摄像头计算目标距离位置。期间利用动轮编码器及PID算法使机器移动至目标前方
    2. 自动贴合：利用TOF距离传感器感知目标距离并利用智能摄像头感应目标水平偏差
    3. 位置微调系统：驾驶员选择瞄准位置时程序会辅助调整机器位置
- **手动阶段精准控制系统**
    1. 动态轨迹纠正系统： 通过IMU获取方向，保证机器移动方向始终与驾驶员参照系方向一致
    2. 从动轮搭配PID纠正算法，确保机器以直线运动
